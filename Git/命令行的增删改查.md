### 增（文件/目录）
#### 文件
```git
<!-- 1. 创建一个空的1.txt文件 -->
touch 1.txt

<!-- 2. 创建一个有内容的1.txt 文件 -->
echo "Hello xiaohe" > 1.txt

<!-- 3. 追加更多的内容到 1.txt 文件 -->
echo "欢迎入坑前端" >> 1.txt
```
#### 目录
```git
<!-- 1. 创建一个空目录（文件夹） -->
mkdir xiaohe
```
### 删（文件/目录）
#### 文件
```git
<!-- 1. 删除 1.txt 文件 -->
rm 1.txt
```
#### 目录
```git
<!-- 1. 删除 xiaohe 目录 -->
rm -r xiaohe

<!-- 2. 强制删除目录（主要用于一个文件夹里有很多的文件，一键删除，没有任何的提示） -->
rm -r xiaohe -f
```

### 改
更改内容的话，我通常会使用命令打开这个文件，然后进行编辑
```git
<!-- 1. 打开 1.txt 文件 -->
start 1.txt
```

<!-- 1. ls代表简单查文件（不包含隐藏文件在内） -->
ls

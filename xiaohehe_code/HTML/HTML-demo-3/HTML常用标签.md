### 常用标签
#### a 标签

（1）href 表示超链接

（2）target 表示在哪个窗口打开,通过中国人习惯在新窗口打开网页。

```
<a href="https://www.baidu.com/" target="_blank">百度一下</a>
```
#### table 标签

```
  </thead>

  <tbody>
  <tr>
   <td>数学</td>

   <td></td>
  </tr>
  </tbody>

  <tfoot></tfoot>
</table>
```
